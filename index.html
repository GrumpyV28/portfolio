Application de Recherche dans l'active directory

Le but de l’entreprise avec cette mission était de pouvoir faire des requêtes avec l’active directory pour en retirer plusieurs informations (Groupe de travaux , permissions et informations professionnelles) Technologie et langage concernés : Utilisation de Visual Studio 2022, Active directory , Windows présentation et XAML J’ai tout d’abord commencé par écrire mon code en Python et dès qu’il a été opérationnel, j’ai pu façonner mon application. L’application possède pour la recherche utilisateur une Search Bar par nom de famille ou prénom. Elle permet donc d’avoir des résultats comme les informations professionnelles (téléphone, e-mail , agence, métier), groupes , sous groupes et permissions.

Requetes API

Le but de l’entreprise avec cette mission était de pouvoir faire des requêtes avec les API de WEBFLEET et HIBOO pour retirer plusieurs informations liées aux différents véhicules en circulation (Position , vitesse...) Technologies et langages concernés : Utilisation de Visual Studio 2022, API WEBFLEET et HIBOO , Excel et base de données . J’ai tout d’abord commencé par faire mon code en python dans VS Code pour terminer ensuite en powershell. Les API sont fonctionnelles avec extraction des données vers une base de données. Mon code permet donc de faire des requêtes API WEBFLEET et Hiboo puis d’envoyer les données vers une base de données afin que la personne qui analyse les données puisse le faire plus aisément et rapidement.
