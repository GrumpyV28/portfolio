Travaux

Application de Recherche dans l'active directory

Le but de cette mission était de pouvoir effectuer des requêtes sur l’Active Directory afin d’en extraire diverses informations (groupes de travail, permissions, informations professionnelles).
Technologies utilisées : Visual Studio 2022, Active Directory, Windows Presentation Foundation (WPF), XAML.
J’ai d’abord écrit le code en Python, puis conçu l’application avec une barre de recherche par nom ou prénom.
Elle affiche les informations professionnelles (téléphone, e-mail, agence, métier), les groupes, sous-groupes et permissions.

Requetes API

L’objectif était d’effectuer des requêtes avec les API de WebFleet et Hiboo pour extraire des données sur les véhicules (position, vitesse, etc.).
Technologies utilisées : Visual Studio 2022, API WebFleet & Hiboo, Excel, base de données.
Le code a été développé en Python (VS Code), puis finalisé en PowerShell.
Les données sont extraites via API puis envoyées vers une base de données, facilitant leur analyse.

Application Mediatek

Contexte :
Mediatek Formation est un projet de gestion de playlists et de formations visant à faciliter l’accès aux informations du site.

Technologies utilisées :
WAMP (serveur local), SonarLint (analyse du code), MySQL (base de données), PHP & Symfony (développement), Git/GitHub (versionning), NetBeans (IDE).

Missions :
Mission 0 : Installation de l’environnement de travail (NetBeans, Git, WAMP), création de la base SQL, clonage du projet via GitHub.
Mission 1 : Nettoyage du code avec SonarLint, ajout de la fonctionnalité pour trier les formations par nombre (croissant/décroissant).
Mission 2 : Ajout de pages pour lister, modifier, supprimer ou ajouter des formations. Gestion des playlists et des catégories. Authentification fonctionnelle avec déconnexion.
Mission 3 : Tous les tests sont opérationnels. Plan de test et PV de recette complétés. Documentation technique et utilisateur conformes aux critères.
Mission 4 : Déploiement du site, base de données et documentation. CGU créée. Sauvegarde journalière automatisée.
